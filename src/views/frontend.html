Creating front-end application for fhirplace

node.js is used for development. Preferred way to install nodejs is nvm (node version manager).

When you have installed node, clone example repository fhirbase-empty-plugin: 

git clone https://github.com/fhirbase/fhirplace-empty-plugin [APP-NAME]

This is a preconfigured application which consists of:

  • fhir.js
  • angularjs
  • coffee
  • less
  • npm
  • bower
  • webpack

You can publish it on any installation of fhirplace server
