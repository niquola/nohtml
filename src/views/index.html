fhirbase

Open source stack for Health IT application development based on FHIR standard

L

fhirbase is an open stack of libraries and services for fast development of healthcare information systems

Huge monolithic systems and vendor locks should become a thing of the past. We believe that open standards and open source software are the future of the Health IT. fhirbase is a collection of open libraries and services for building healthcare information systems. We use modern open source technologies as a technical basis (Linux, PostgreSQL, HTML, CSS, JavaScript, Java, Clojure) and open FHIR standard as semantic foundation. All components form full stack from data layer (fhirbase) to web and mobile applications (fhir.js, formstamp).

fhirbase is a part of growing FHIR ecosystem and most of its components can be used separately with FHIR compliant technologies.

User/Client Layer

User Interface (front-end) applications are built using HTML, CSS and JavaScript using REST API and WebSockets.

Application Layer

Business logic, integrations, messaging (HL7 v2, CDA etc), security rules

Data Layer

Data consistency, custom queries, analytics and reporting. Terminology and dictionary databases.
